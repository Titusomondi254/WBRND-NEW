## 💬 Messaging Guide

### Agent Sending a Message:
1. Go to Agent Dashboard
2. Scroll to **Messages** section
3. Click **📧 Send Message** button
4. Fill in:
   - Subject (e.g., "Question about property")
   - Message (detailed text)
5. Click **Send Message**
6. Admin will receive it and can reply

### Agent Replying to Admin:
1. See message in Messages section
2. Click **Reply** button on any message
3. Type your response
4. Click **Send Reply**

### Admin Sending Message:
1. Go to Admin Messages page
2. Select agent from left sidebar
3. Scroll to bottom
4. Fill in Subject and Message
5. Click **Send Message**
6. Agent gets notification

## 📊 Understanding the Data

### Leads Section:
- Shows customers who requested viewings/consultations
- **Name**: Customer name
- **Contact**: Email & Phone
- **Type**: Service requested (Viewing, Valuation, etc.)
- **Status**: Current stage (Pending, Completed, Cancelled, Scheduled)

### Properties Section:
- **Uploaded**: All properties you've added
- **Successful Connections**: Properties with completed deals
- **Rejected**: Properties not approved by admin

### Clients Section:
- **Connected**: Clients who bought/rented from you
- **Happy 😊**: Gave positive feedback (GREEN)
- **Complained 😞**: Gave negative feedback (RED)
- **Failed to Pick**: Abandoned viewings (ORANGE)

### Viewings Section:
- **Assigned**: Total viewing requests sent to you
- **Completed**: Viewings you finished (GREEN)
- **Pending**: Viewings waiting to be done (ORANGE)
- **Expired**: Viewings past due date (RED)

### Delivery Groups Section:
- Shows teams managing deliveries
- **Total Bookings**: Number of jobs assigned to group
- **Completed**: Jobs finished successfully
- **Pending**: Jobs in progress
- **Rescheduled**: Jobs moved to different date
- **⭐ Rating**: Star rating based on customer feedback (1-5 stars)

### Tasks Section:
- Shows progress across all service types
- **Categories**: 11 types of services
- **Green bars**: Completed tasks
- **Orange bars**: Pending/incomplete tasks

### Messages Section:
- **Red badge**: Shows count of unread messages
- **Send Message**: Compose new message to admin
- **Reply**: Respond to specific message
- Shows: Title, Message preview, Sender, Timestamp

## ❓ FAQ

**Q: How often does the data update?**
A: Data is pulled from the database when you load the page. Refresh to see latest data.

**Q: Can I edit information on the dashboard?**
A: No, dashboard is view-only. Use specific management pages (like "Manage Properties") to edit.

**Q: How do I delete a message?**
A: Messages are archived automatically after 30 days. Currently no delete function.

**Q: Can I export the dashboard data?**
A: Currently exporting is not available. Coming in future updates.

**Q: How many messages can I store?**
A: All messages are stored. Latest 20 are shown on dashboard.

**Q: Who can see my messages?**
A: Only you and the admin can see your messages. Secure and private.

**Q: What if I'm not receiving messages?**
A: Check that admin is sending to your account. Admin uses your user ID.

**Q: How do delivery group ratings work?**
A: Stars are based on client feedback/reviews. More positive reviews = higher rating.

## 🆘 Troubleshooting

**Problem**: Dashboard not loading
- **Solution**: Refresh page or clear browser cache

**Problem**: No data showing
- **Solution**: Make sure you have consultations/properties in your account

**Problem**: Messages not sending
- **Solution**: Fill all fields (Subject & Message), then try again

**Problem**: Delivery groups not showing
- **Solution**: Make sure groups are assigned to you in mover system

**Problem**: Tasks chart not displaying
- **Solution**: Ensure JavaScript is enabled in browser
